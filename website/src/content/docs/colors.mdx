## Palette structure

Coldark uses three palettes:

- a shared set of blue-grey shades used by both variants
- a light accent palette for **Coldark Cold**
- a dark accent palette for **Coldark Dark**

## Common shades

| Token       | Hex       | Role                                                                   |
| ----------- | --------- | ---------------------------------------------------------------------- |
| `coldark00` | <ColdarkHex token="coldark00" /> | Lightest background for elevated panels, widgets, and active surfaces. |
| `coldark01` | <ColdarkHex token="coldark01" /> | Soft bright surface and bright foreground support.                     |
| `coldark02` | <ColdarkHex token="coldark02" /> | Secondary surface and light foreground support.                        |
| `coldark03` | <ColdarkHex token="coldark03" /> | Selections, borders, and muted emphasis.                               |
| `coldark04` | <ColdarkHex token="coldark04" /> | Comments, placeholders, and subdued text.                              |
| `coldark05` | <ColdarkHex token="coldark05" /> | Strong surface for widgets, highlights, and embedded panels.           |
| `coldark06` | <ColdarkHex token="coldark06" /> | Main dark foreground or background, depending on the variant.          |
| `coldark07` | <ColdarkHex token="coldark07" /> | Deepest structural background for heavy UI chrome.                     |

## Accent slots

| Token       | Coldark Cold | Coldark Dark | Role                                                                |
| ----------- | ------------ | ------------ | ------------------------------------------------------------------- |
| `coldark08` | <ColdarkHex token="coldark08" variant="cold" /> | <ColdarkHex token="coldark08" variant="dark" /> | Red for errors, deletions, invalid states, and deprecated items.    |
| `coldark09` | <ColdarkHex token="coldark09" variant="cold" /> | <ColdarkHex token="coldark09" variant="dark" /> | Orange for warnings, keywords, storage, and pseudo-elements.        |
| `coldark10` | <ColdarkHex token="coldark10" variant="cold" /> | <ColdarkHex token="coldark10" variant="dark" /> | Yellow for modified states, hints, booleans, and attribute names.   |
| `coldark11` | <ColdarkHex token="coldark11" variant="cold" /> | <ColdarkHex token="coldark11" variant="dark" /> | Green for additions, strings, CSS colors, and link text.            |
| `coldark12` | <ColdarkHex token="coldark12" variant="cold" /> | <ColdarkHex token="coldark12" variant="dark" /> | Cyan for parameters, tags, CSS variables, and embedded punctuation. |
| `coldark13` | <ColdarkHex token="coldark13" variant="cold" /> | <ColdarkHex token="coldark13" variant="dark" /> | Blue for buttons, badges, info, classes, variables, and headings.   |
| `coldark14` | <ColdarkHex token="coldark14" variant="cold" /> | <ColdarkHex token="coldark14" variant="dark" /> | Purple for functions and CSS classes or IDs.                        |
| `coldark15` | <ColdarkHex token="coldark15" variant="cold" /> | <ColdarkHex token="coldark15" variant="dark" /> | Magenta for untracked states, regex, escapes, and CSS values.       |
